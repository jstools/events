Installation
------------
```.sh
npm install jstools-events --save
```
or
```.sh
bower install jn-events --save
```
Usage
-----
```.js
var obj = new Events();

obj.$$on('foo', function () {
  flag = true;
});

obj.$$trigger('foo');
```
